# 🎮 CodeVibe Academy | Learn to Code by Doing 🚀

> An immersive, gamified, and 3D-styled (Cyberpunk/Dark Mode) learning platform designed to demystify programming. Learn HTML, CSS, JavaScript, and VS Code shortcuts hands-on!

![Project Preview](link-to-your-image-or-gif-here.gif)

Reading endless theory can be unmotivating for beginners. **CodeVibe Academy** was created for students and young adults who want to learn how to code in the most engaging way possible: by doing it. Featuring 50 practical challenges based on essential Cheat Sheets, the platform turns your screen into a real code simulator.

---

## 🚀 Features

* **Immersive Environment:** Cyberpunk-style design with 3D elements, Glassmorphism, and neon highlights.
* **50 Practical Challenges:** 
    * 🟦 10 HTML Challenges (Structure and Semantics).
    * 🟧 10 CSS Challenges (Flexbox, Grid, and Styling).
    * 🟨 10 JavaScript Challenges (Logic, DOM, and Arrays).
    * 💻 20 VS Code Challenges (Shortcuts and Productivity).
* **Integrated Simulator:** An interactive terminal on one side and a code editor on the other. Immediate feedback on every answer!
* **Gamification (XP & Levels):** Earn points for correct answers and get motivational hints when you make a mistake.
